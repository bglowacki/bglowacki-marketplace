## Getting Started

### For Users

```bash
# Add marketplace
claude plugins add-marketplace github:bglowacki/bglowacki-marketplace

# Install plugin
claude plugins install observability@bglowacki-marketplace

# Collect usage data
/observability-usage-collector
```

### For Developers

```bash
# Setup
cd observability
uv sync

# Run tests
uv run pytest tests/

# Test hook
echo '{"session_id":"test","cwd":"/tmp"}' | uv run hooks/generate_session_summary.py
```

See [Development Guide](./development-guide.md) for full setup instructions.

## Data Flow
