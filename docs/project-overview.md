# bglowacki-marketplace - Project Overview

## Executive Summary

A personal Claude Code plugins marketplace hosting the **observability** plugin for session analysis and usage insights. The plugin analyzes Claude Code session logs to help users understand their tool usage patterns, identify missed opportunities, and optimize their workflows.

## Project Purpose

This project serves two functions:

1. **Marketplace**: A registry (`marketplace.json`) for distributing Claude Code plugins
2. **Plugin Host**: Contains the `observability` plugin for session analytics

### Observability Plugin Features

- Generates session summaries saved to `~/.claude/session-summaries/`
- Tracks tool outcomes (success/failure/interrupted)
- Monitors workflow stages (brainstorm, plan, implement, test, review, commit)
- Analyzes skill/agent usage patterns
- Context efficiency tracking (compaction metrics)
- macOS notifications on session end

## Architecture Type

**Plugin/Hook Architecture** - The plugin integrates with Claude Code through:

- **Stop Hook**: Runs on session end to generate summaries
- **Skills**: User-invocable workflows (`/observability-usage-collector`)
- **Agents**: AI-powered analysis (`usage-insights-agent`)
- **Commands**: Orchestration shortcuts

## Installation

```bash
# Add marketplace

# Install observability plugin
claude plugins install observability@bglowacki-marketplace
```
